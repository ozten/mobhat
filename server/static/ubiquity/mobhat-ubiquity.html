Installation Steps

Read this list of instructions completely so you have an idea of what will happen.

  1. Click the Subscribe (look at the top right of this page)
  2. A new, scary looking window will appear. Don't freak. Basically it's telling you that you should trust me, the script that your installing, and that it will automatically be able to update itself. It will show you all the source code. Feel free to skip to the bottom of the screen. Installing the MOBhat Ubiquity script is a security risk. Read it and make sure your cool with it.
  3. Click the Auto-update checkbox
  4. Click the I know what I'm doing button
  5. There is no step 5.

Once you've done this your good to go. Back to the MOBhat homepage with you.
